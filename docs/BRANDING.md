# FeTreeOS — 브랜딩 & 런칭 계획

## 비전

알고리즘과 SNS, AI 생성형이 거세하는 인간의 창의성과 주체성 —
숏폼과 양산형 콘텐츠의 늪에서 벗어나, 소비자의 창작자화를 이끌어내는 인터랙티브 공간.

FeTreeOS는 포트폴리오가 아니라 **경험 환경**이다.
방문자는 관람하지 않고, 들어와서 만들고, 열고, 정리하고, 머문다.
본래의 핵심동력은 만든 이의 욕심이지만, 결과적으로는 인류의 방주가 되었으면 하는 바람.

**기능 판단 기준**: "이것이 소비자의 창작자화에 기여하는가?"

이름의 기원인 Digital Exodus(디지털 탈출)가 이 취지를 관통한다.

### 콘셉트

노스탤지어의 상징 — 닌텐도DS, 플래시, Windows XP 시대를 **재해석**하는 것.
복제가 아닌 오마주. 그 시절의 느낌을 따르되, 에셋은 직접 만든다.

### 에셋 원칙

- 음악, 애니메이션, 이미지 등 모든 리소스를 **자체 제작**
- 외부 에셋 의존 없음 → 법적 리스크 원천 차단, 동시에 차별점
- XP 배경화면 오마주, DS 부팅음 느낌의 사운드 등은 직접 창작 → 저작물로서 보호됨

### 창작자화 — 커스터마이징을 넘어 창작으로

FeTreeOS의 커스터마이징은 "테마를 골라서 적용"이 아니라 **OS 안에서 직접 만드는 것**:

| 영역 | 사용자 창작 | 웹 기술 |
|------|-----------|---------|
| 배경화면 | 벡터 드로잉, 페인팅 | Canvas API, SVG |
| 효과음 | 실시간 녹음 | MediaRecorder API |
| 테마 | 색상/레이아웃 조합 | CSS 변수 |
| 폰트 | 업로드 또는 선택 | FontFace API |
| 데스크탑 펫 | 스프라이트 조립/그리기 | Canvas 애니메이션 |

만든 에셋은 Supabase Storage에 저장 → 다른 사용자와 공유 가능 → 앱 카탈로그/마켓 구조로 확장.
커스터마이징 자체가 콘텐츠가 되고, 소비자가 창작자가 되는 순환.

## 이름 확정: FeTreeOS

### 네이밍 경위

1. **Digital Exodus → Exdos**: "Exodos"라는 기존 사이트와 충돌
2. **TreeOS**: 채널명 FeTree의 Tree + OS. 이미 동명 소프트웨어 존재
3. **TryOS (트리오스)**: "일단 해봐" + OS. 발음에 Tree 뉘앙스 유지. 그러나 GitHub에 소규모 동명 프로젝트 다수 (pooolman/tryos, Rudhra0811/tryOS, TryOS-Labs org 등)
4. **FeTreeOS**: 채널명 FeTree + OS. 검색 결과 **완전히 비어있음** — 웹, GitHub, npm, 도메인 전부 미사용. 채널 브랜딩과 직결

### 채택 이유

- 채널(FeTree) 방문자가 즉시 연결 가능
- 발음 "페트리오에스" — 이미 알려진 채널명 + OS
- 도메인 .dev/.com/.io 전부 미등록 — 완전한 독점 가능
- 상표 등록 기록 없음 (USPTO, KIPRIS 기준 2026-04-27)

### 의미

- **Fe**: 철(Iron) — 단단함
- **Tree**: 나무 — 성장, 뿌리
- **OS**: Operating System — 웹 기반 가상 OS
