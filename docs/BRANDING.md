# FeTreeOS — 브랜딩 & 런칭 계획

## 이름 확정: FeTreeOS

### 네이밍 경위

1. **Digital Exodus → Exdos**: "Exodos"라는 기존 사이트와 충돌
2. **TreeOS**: 채널명 FeTree의 Tree + OS. 이미 동명 소프트웨어 존재
3. **TryOS (트리오스)**: "일단 해봐" + OS. 발음에 Tree 뉘앙스 유지. 그러나 GitHub에 소규모 동명 프로젝트 다수 (pooolman/tryos, Rudhra0811/tryOS, TryOS-Labs org 등)
4. **FeTreeOS**: 채널명 FeTree + OS. 검색 결과 **완전히 비어있음** — 웹, GitHub, npm, 도메인 전부 미사용. 채널 브랜딩과 직결

### 채택 이유

- 채널(FeTree) 방문자가 즉시 연결 가능
- 발음 "페트리오에스" — 이미 알려진 채널명 + OS
- 도메인 .dev/.com/.io 전부 미등록 — 완전한 독점 가능
- 상표 등록 기록 없음 (USPTO, KIPRIS 기준 2026-04-27)

### 의미

- **Fe**: 철(Iron) — 단단함
- **Tree**: 나무 — 성장, 뿌리
- **OS**: Operating System — 웹 기반 가상 OS
